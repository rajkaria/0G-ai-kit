# Error codes

Every error `0gkit-*` throws carries a stable `code` from this list plus a `helpUrl` that links back here. Click a code for cause, fix, and a minimal example.

## CONFIG

- [`CONFIG_MISSING_ENV`](/errors/CONFIG_MISSING_ENV) ‚Äî Missing required environment variable
- [`CONFIG_INVALID_NETWORK`](/errors/CONFIG_INVALID_NETWORK) ‚Äî Unknown network name
- [`CONFIG_INVALID_ADDRESS`](/errors/CONFIG_INVALID_ADDRESS) ‚Äî Not a valid 0x address
- [`CONFIG_INVALID_ARGUMENT`](/errors/CONFIG_INVALID_ARGUMENT) ‚Äî Invalid argument passed to a primitive

## WALLET

- [`WALLET_NO_PRIVATE_KEY`](/errors/WALLET_NO_PRIVATE_KEY) ‚Äî No private key derived from mnemonic
- [`WALLET_KMS_SIGN_FAILED`](/errors/WALLET_KMS_SIGN_FAILED) ‚Äî AWS KMS Sign returned no signature
- [`WALLET_KMS_PUBKEY_FAILED`](/errors/WALLET_KMS_PUBKEY_FAILED) ‚Äî AWS KMS GetPublicKey failed
- [`WALLET_BAD_DER_SIGNATURE`](/errors/WALLET_BAD_DER_SIGNATURE) ‚Äî Malformed DER signature from KMS
- [`WALLET_NO_CONNECTOR`](/errors/WALLET_NO_CONNECTOR) ‚Äî No wagmi connector available
- [`WALLET_CHAIN_MISMATCH`](/errors/WALLET_CHAIN_MISMATCH) ‚Äî Signer chainId doesn't match active chain

## CHAIN

- [`CHAIN_RPC_UNREACHABLE`](/errors/CHAIN_RPC_UNREACHABLE) ‚Äî RPC endpoint unreachable
- [`CHAIN_RPC_TIMEOUT`](/errors/CHAIN_RPC_TIMEOUT) ‚Äî RPC call timed out
- [`CHAIN_TX_REVERTED`](/errors/CHAIN_TX_REVERTED) ‚Äî Transaction reverted on-chain
- [`CHAIN_TX_TIMEOUT`](/errors/CHAIN_TX_TIMEOUT) ‚Äî Transaction did not confirm in time
- [`CHAIN_INSUFFICIENT_FUNDS`](/errors/CHAIN_INSUFFICIENT_FUNDS) ‚Äî Account has insufficient funds
- [`CHAIN_NONCE_TOO_LOW`](/errors/CHAIN_NONCE_TOO_LOW) ‚Äî Nonce already used

## STORAGE

- [`STORAGE_QUOTA_EXCEEDED`](/errors/STORAGE_QUOTA_EXCEEDED) ‚Äî Storage quota exceeded
- [`STORAGE_UPLOAD_FAILED`](/errors/STORAGE_UPLOAD_FAILED) ‚Äî Upload to 0G storage failed
- [`STORAGE_DOWNLOAD_FAILED`](/errors/STORAGE_DOWNLOAD_FAILED) ‚Äî Download from 0G storage failed
- [`STORAGE_ROOT_NOT_FOUND`](/errors/STORAGE_ROOT_NOT_FOUND) ‚Äî Storage root not found on network
- [`STORAGE_ROOT_MISMATCH`](/errors/STORAGE_ROOT_MISMATCH) ‚Äî Computed root doesn't match expected
- [`STORAGE_INVALID_BYTES`](/errors/STORAGE_INVALID_BYTES) ‚Äî Storage input bytes empty or invalid

## COMPUTE

- [`COMPUTE_PROVIDER_UNREACHABLE`](/errors/COMPUTE_PROVIDER_UNREACHABLE) ‚Äî 0G Compute provider unreachable
- [`COMPUTE_NO_PROVIDER`](/errors/COMPUTE_NO_PROVIDER) ‚Äî No provider serving the requested model
- [`COMPUTE_INFERENCE_FAILED`](/errors/COMPUTE_INFERENCE_FAILED) ‚Äî Inference call returned an error
- [`COMPUTE_BAD_ATTESTATION`](/errors/COMPUTE_BAD_ATTESTATION) ‚Äî TEE attestation verification failed
- [`COMPUTE_BUDGET_EXCEEDED`](/errors/COMPUTE_BUDGET_EXCEEDED) ‚Äî Inference cost exceeded budget

## DA
