# `create-0g-app`

`create-0g-app` was the original planned npm-create package, but that npm name
is held by another publisher. Use the working public initializer instead:

```bash
npm create 0gkit-app@latest my-app
cd my-app
0g dev          # in another terminal — starts the local devnet
npm run dev     # runs the scaffolded app
```

See the [`create-0gkit-app` reference](/getting-started/create-0gkit-app) for
flags, template descriptions, and troubleshooting.
